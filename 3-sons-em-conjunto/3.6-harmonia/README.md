# 3.6 Harmonia

* Díade: duas notas (alturas) tocadas simultaneamente ou sequencialmente. É um termo pouco usado na música tonal, mas mais usado na música não tonal.
* Tríade: três notas (alturas) tocadas simultaneamente ou sequencialmente.
